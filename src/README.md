# Lift Operator 2022

* Extreme lift-operating action.
* Play as a lift operating software, managing payloads of passengers.

### Controls
* `left-click` to select a floor from the panel.
* `F1` to toggle fullscreen.
* `F3` toggle debug (FPS).
* `Escape` to quit.
* `p` to pause background music.
